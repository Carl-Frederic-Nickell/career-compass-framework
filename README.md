# Career Compass Framework

A structured, AI-assisted framework for planning career transitions and professional development.

## Overview

Career Compass helps you create a comprehensive, market-validated career strategy through:
- **85 self-assessment questions** covering skills, values, goals, and constraints
- **AI-powered career path discovery** with 3-5 tailored recommendations
- **Market research integration** with real salary data, job market analysis, and industry trends
- **90-day action plans** with specific milestones and success metrics

## Quick Start

1. Complete `2-QUESTIONS.md` (2-4 hours)
2. Run `3-DISCOVERY-PROMPT.md` with an AI assistant
3. Run `4-FINAL-STRATEGY-PROMPT.md` with an AI that has web search
4. Execute your 90-day action plan

## Requirements

- AI assistant with web search capabilities (Claude, ChatGPT-4, Perplexity Pro)
- 4-6 hours total for the complete process
